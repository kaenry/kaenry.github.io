让·爱

发表于 2016-06-20 更新于 2021-03-09 分类于 生活 Valine：
本文字数： 456 阅读时长 ≈ 1 分钟

我并不伟大，
但我还是学者那些韩剧里的男主角， 
虽然爱你，但放你走， 
让你去追逐你的幸福，然后对自己说， 
你一定要幸福。

却不曾想，让爱，
是这样痛苦与煎熬， 那样伤，
那些天，我说不出一个字 
思念如潮水般袭来， 不断冲刷着我的心。

你那明净的双眼，快要把我灼伤， 
终于还是决堤 
因为又一次见到你， 
我不顾一切的向你奔去。

没有看见机车，
没有看见行人，
看见 你的他就在你的身边， 
但我的眼里此刻只有你。

爱，不知所以

发表于 2016-06-20 更新于 2021-03-09 分类于 生活 Valine：
本文字数： 166 阅读时长 ≈ 1 分钟

每一个流星划过的夜晚，
每一个痴狂你的深夜， 
想念你 
你那嘟着小嘴生气的样子， 
你那倔强的任性， 
你那可爱的天真， 
你那骨子里的骄傲， 
你那不讲理的胡搅蛮缠， 
你眼睛里的忧郁， 
你眸子里的纯净， 
你体贴的温柔， 
你的善解人意， 
爱着你，我不知所以， 
你贴着我的发梢说“我爱你”， 
你突然转过身地问“爱不爱我”， 
我吻你你总是躲， 
你吻我我却来不及躲

摘自未出版某小说

默爱。女子

发表于 2016-06-20 更新于 2021-03-09 分类于 生活 Valine：
本文字数： 140 阅读时长 ≈ 1 分钟

如果我恨你， 
我会斩钉截铁地对你说，
可如今我爱你，
对人类如此含糊的语言， 
我却信不过。

你愿它化作一声呼唤， 
来自是神的心底， 
可它还没出胸膛和喉咙，
灼热的激流早已有气无力。 
我本是一个涨满的池塘， 
可对你却像干涸的泉眼一样， 
一切都由于我痛苦的沉默， 
它的沉默胜过死亡。

摘自某年某杂志（真是忘了）

灰姑娘的姐姐

发表于 2016-06-20 更新于 2018-06-01 分类于 生活 Valine：

那个人，不看难受，
看了也难受 在旁边是难受，
不在了也难受 对着我笑难受，
对着别人笑也难受 
叫着我的名字，难受 
不叫的话也很难受 
如果我没有一下子陷进地里 
总是，会难受的 
可是，还是在旁边比较好 
即使看着，讨厌那个人 也比起他不在，好得多了…

摘自韩剧《灰姑娘的姐姐》
